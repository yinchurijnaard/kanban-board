# Kanban Board

## How it's made

This project is built with [Vite](https://vite.dev) + [React](https://react.dev), [Tailwind CSS](https://tailwindcss.com) and [daisyUI](https://daisyui.com), deployed on [Vercel](htts://vercel.com). It's designed to be a PWA with CRUD functionality through the use of local storage.

## Live demo

Click on the following link to try out the Kanban Board for yourself: https://kanban-board-lime-iota.vercel.app/.
